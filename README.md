The library for basic mathematics.

## Setup
To install Blaze, run the following command in the Package Manager Console on Visual Studio:

```
Install-Package Blaze
```

[NuGet Gallery | Blaze](https://www.nuget.org/packages/Blaze/)

## Target Frameworks
- .NET Standard 2.0
- .NET Framework 4.5
